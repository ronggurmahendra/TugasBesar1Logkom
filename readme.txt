Game ini adalah permainan petualangan RPG yang berfokus pada battle dan gacha dengan bahasa ProLog.
Pemain akan mengalahkan musuh untuk menyelesaikan quest, lalu akan bertambah kuat dengan mendapatkan item dan naik level. 
Jika health player habis maka akan mati dan kembali ke menu
Tujuan akhirnya mengalahkan naga untuk menyelamatkan dunia!

TULIS command 'help.' untuk membuka pilihan jika terjadi kebingungan. 
Command:
help. : memperlihatkan list command pada state tersebut
//Menu//
start. : memulai permainan
load. : load permainan
//Eksplorasi//
w. : bergerak ke arah utara/atas.
a. : bergerak ke arah kiri/barat.
s. : bergerak ke arah selatan/bawah.
d. : bergerak ke arah timur/kanan.
map. : memperlihatkan peta
status. : status pemain
shop. : masuk ke shop 
save. : simpan kondisi 
quit. : keluar
swap_weapon. : tukar senjata
swap_armor. : tukar armor
swap_accessory. : tukar accessory
//Battle//
attack. :
special_attack. : melakukan special attack, cooldown 3 turn
usePotion. : heal dengan potion
run. : lari dari battle
//Shop//
healthpotion. : beli health potion
gacha. : mendapatkan barang secara acak, ada kemungkinan special reward!
sell. : jual barang untuk mendapat gold

Cara Bermain:
1. Buka main.pl dengan menggunakan GNU ProLog
2. Ketik command 'start.' untuk memulai dari awal, perintah 'load.' jika ingin melanjutkan permainan.
3. Ikuti petunjuk permainan jika tidak ingin error
3. Game berakhir dan akan keluar ke menu, akan muncul pesan 'You Died' atau 'You Win'
